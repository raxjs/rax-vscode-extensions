# Rax

This VS Code extension adds support for [Rax](https://rax.js.org/).

## 1. Rax Snippets
### Snippets

| Snippet | Renders                     |
| ------- | --------------------------- |
| `ir`    | Import Rax                  |
| `irc`   | Import Rax / Component      |
| `irh`   | Import Rax / Hooks          |
| `cc`    | Class Component             |
| `fc`    | Function Component          |
| `ush`   | Use State Hook              |
| `ueh`   | Use Effect Hook             |
| `umh`   | Use Memo Hook               |

## 2. Create Rax
### Usage

1. `cmd + shift + p` (Windows: `ctrl + shift + p`)
2. Input `rax create`
3. Enter
